# 🧺 Laundry Order Management System

## 💼 Problem
Managing laundry orders manually leads to errors, delays, and poor tracking.

---

## ✅ Solution
This system digitizes order tracking, billing, and workflow management for dry cleaning businesses.

---

## 📊 Features
- Order creation & tracking  
- Status updates  
- Automated billing  
- Dashboard insights  

---

## 🤖 Development Approach
Built using AI-assisted tools to accelerate development and improve efficiency.

---

## 🛠️ Stack
React | Node.js | Express | REST API
