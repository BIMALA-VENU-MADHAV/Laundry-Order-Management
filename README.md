# 🧺 Laundry Order Management System

## 📌 Description
A full-stack application designed for dry cleaning businesses to manage orders, track status, calculate billing, and monitor daily operations through a dashboard.

---

## 🛠️ Tech Stack
- Frontend: React.js, CSS  
- Backend: Node.js, Express.js  
- Deployment: Vercel (Frontend), Render (Backend)  

---

## ✨ Features
- Create and manage orders  
- Track order status  
- Automated billing calculation  
- Dashboard with business insights  

---

## ⚡ Highlights
- Built using an AI-assisted development approach  
- Focused on scalability and clean architecture
